Session 1 — the piece

Pentagon shōgi-piece glyph: geometry and both faces. Front face 歩 in sumi; promoted face と in gold.
Ink values on this page are placeholders — sumi #1a1a1a, gold #b8860b, washi #f7f4ec — for display only. Real values are session 2.

Recommendation

A — true koma ratios, sumi keyline, character at 0.42H — with B's solid silhouette frozen as the sub-24px form.

Reasons. The true koma proportions (flared sides, off-square shoulders) are what make this the shōgi pawn and not a generic pentagon — C and D read as "home" or "arrow up" icons the moment the flare goes. The keyline-and-ink-character treatment is the woodblock production logic itself: a keyblock line printed on paper, the character in the same sumi ink. B inverts that (character as paper cut from an ink field), which is honest but heavier, and D's oversized character crowds the shoulders.

The honest finding at 16 px: no face character survives, and a 0.04H keyline scales to under one pixel. So the small-scale form is not a smaller drawing of the large one — it is a second inking of the same block. Below 24 px rendered height the piece drops its character and prints solid: sumi fill for the front face, gold fill for the promoted face. One geometry, two inkings, chosen by scale. Gold-as-fill at that scale is state marking (a closed ho), not emphasis, so it stays inside the gold rule.

The scale rule demonstrated: 160 / 48 / 32 px keyline form; 16 px solid form, front (sumi) and promoted (gold).
