Session 9 — the flip

hosystem · one question: where promotion fires, and how the flip moves.

The name is the shōgi pawn. It promotes by physically flipping over to reveal と in gold. On the site the flip is the state change — the one moment the restraint breaks into something kinetic, because a real state changed. Gold is state only: the flip never fires as hover, scroll effect, or emphasis.

Decisions carried into every variant

  • Axis correction. A koma promotes by tipping end-over-end toward the opponent — rotation about the horizontal axis. Spinning about the vertical axis is what a captured piece does when it changes sides. So the 2D flip is a vertical squash (scaleY through zero), not the horizontal scale the prompt sketched. The pentagon is left-right symmetric, so only the vertical squash encodes the real move.
  • Superseded never flips. Supersession is recession — gold@1.00 → gold@0.40, opacity not promotion (§2: "recession is opacity"). The flip means one thing: promotion. A superseded record arrives already-flipped, receded. Every demo shows this.
  • One committed move. 160 ms total, §4's easing across the whole flip. On cubic-bezier(0.2, 0, 0, 1), progress 0.5 lands at exactly 32 ms of clock time — the collapse commits in 32 ms, the promoted face settles over the remaining 128 ms. Accelerate, land. Adjacent state tints (node stroke) cut at the 32 ms zero-crossing — a cut, not a fade. No wobble, no bounce, no spin.
  • Queue on the beat. Co-triggered flips fire in encounter order on §4's 380 ms interval. Marks already in view at page arrival start one interval after the page's in-step ends.
  • Demo staging. Production triggers are IntersectionObserver on scroll; the replay buttons stage the encounter. Where a flip is pending, the mark rests on its front face (歩, full sumi — current ink) until it fires. Value text ("closed") always prints full sumi throughout — labels never lie, only the mark moves.

Recommendation — D, with A's geometry

D. The 2D squash, firing under the news rule.

  • The governing test, taken literally. The flip breaks the site's restraint "precisely because a real state changed." Only D makes that sentence true. A and B re-stage a month-old closure for every new session or reader — theater about a change that didn't happen on the reader's watch. C never breaks restraint at all and forfeits the site's namesake move. D fires the flip exactly when the framework actually moved: a ho closed between your visits. First visit — you weren't there for any of it — is rendered honestly: everything already promoted, nothing staged. C is D's degenerate case, so D contains C's restraint.
  • It scales with the corpus. A mature arc holds a dozen closed hos. Under A or B, every history page greets a first view with a flip storm — twelve promotions re-enacted in sequence, 4.5 seconds of gold motion. Under D the flip stays rare, so it stays meaningful. And the colophon — the site's own live record — keeps closing hos as the site is built, so returning readers genuinely meet it.
  • Grammar consistency with §7. Session 7 froze that reader state never earns gold ("met is reader state, not framework state"). B keys a gold event to pure reader state — you hadn't seen it yet. D keys it to a framework transition; the witness log only decides whether that transition is news.
  • Squash over depth. Print logic: flat fields, no object floating off the paper. The perspective flip introduces foreshortening — a 3D vocabulary the site has nowhere else and would use exactly once. The vertical squash is the woodblock rendering of the koma's true move: end-over-end, toward the opponent.
